### SUpabase

- go to project
- go to Build out your database click on [About database](https://supabase.com/docs/guides/database)
- on the left select Connecting to your database
- on the right Pooler transaction mode
- then Get your project's Transaction pooler connection string from your project dashboard by clicking [Connect](https://supabase.com/dashboard/project/_?showConnect=true&method=transaction)
- click on project name that apperars
